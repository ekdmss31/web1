JavaScript란 무엇인가?

JavaScript

java script를 이용해서 버튼을 누르면 body에 style을 추가해주는 기능을 구현할 것이다. night 버튼을 누르면 body의 스타일을 background를 black, 글씨를 white로 변경하도록 하는 것이다. 이를 사용자와 상호작용을 통해 사용자가 버튼을 누르면 변화가 되게 하여야 하기 때문에 이것을 java script로 구현하는 것이다. javascript는 이러한 기능을 가지고 있다. javascript는 html 위에 동작한다. 숫자는 number이고, 변수 선언은 var을 이용한다. style을 동적으로, 상호작용에 의해 넣기 위해서 javascript를 이용하는 것이다!! js는 사용자와 상호작용하기 위해 만든 언어이다. html이나 css는 사용자와 상호작용이 되지 않아서 js를 통해 상호작용할 수 있다. 조건에 따라 다른 동작을 이용하고 싶을 때 사용하는 것이다. for이나 while같은 조건문은 조건에 따라 순서대로가 아닌 다른 순서의 기능들이 실행될 수 있다.

리팩토링이란 코딩을 하고 나면 비효율적으로 나올 수 있는데, 코드를 효율적으로 만들어 가독성을 높이고 중복성을 낮춰 유지보수를 쉽게 하기 위해 개선하는 것을 리팩토링이라고 한다. 즉, 비효율적인 것을 제거하는 것을 리팩토리라고 한다.

원래 사용했던 this는 웹브라우져를 가리킨다. 따라서 input을 가리키기 위해 매개변수를 이용한다. input에서 this를 이용해서 버튼을 가리키게 하면, this값이 function의 매개변수로 들어가서 함수를 실행시킨다.
