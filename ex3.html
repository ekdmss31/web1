JavaScript란 무엇인가?

java script를 이용해서 버튼을 누르면 body에 style을 추가해주는 기능을 구현할 것이다. night 버튼을 누르면 body의 스타일을 background를 black, 글씨를 white로 변경하도록 하는 것이다. 이를 사용자와 상호작용을 통해 사용자가 버튼을 누르면 변화가 되게 하여야 하기 때문에 이것을 java script로 구현하는 것이다. java script는 이러한 기능을 가지고 있다. javascript는 html 위에 동작한다. 숫자는 number이고, 변수 선언은 var을 이용한다. style은 css의 property라는 속성이다.
